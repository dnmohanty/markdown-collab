# Real-Time Markdown Editor

A collaborative, real-time Markdown editor built with React, Vite, and Tailwind CSS.

## Features Built So Far (Day 1)
* Initialized React application using Vite.
* Configured Tailwind CSS v4 for styling.
* Implemented a sleek, dark-themed UI.
* Integrated CodeMirror 6 for the core text editor with markdown syntax highlighting.
* Set up React state to capture and track keystrokes in real-time.

## How to Run Locally
1. Clone this repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local server.
